# 关于Aimwidget

我们整理一系列基于Taro开发的插件（我们称为Widget），内容包括大语言模型应用、Agent、趣味性小游戏、营销小场景，形式上建议单页面，均采用Taro框架开发

# 开发相关

## 步骤

按照项目实际需要，修改以下各文件

- `app.config.js` 
```
'pages/sample/index'
```
- `pages/sample/index.jsx`
```
import { postSamples } from '../../api/sample'
```

## 运行

以微信为例

```
yarn dev:weapp
```
